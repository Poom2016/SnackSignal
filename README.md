# Snack Signal

A food expiration date tracker built with React, Tailwind CSS, and Babel standalone.

## Features

- Calendar view to track food expiration dates
- Color-coded warnings (green/orange/red) based on how soon items expire
- 6 food categories: Meat & Fish, Dairy, Veggie & Fruit, Snack, Drink, Other
- Browser notifications for items expiring tomorrow
- LocalStorage persistence (no backend required)
- Responsive design (mobile and desktop)

## Usage

Open `index.html` in a browser or deploy via GitHub Pages.

## Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Babel Standalone (for in-browser JSX)
- LocalStorage for data persistence
- Web Notifications API
